## 說明
因為未接觸過前後端分離，又在因緣巧合下看到springboot + vue的相關技術文章，因此透過此專案作為練習

## Backend
本專案以 SpringBoot 作為核心框架，搭建了 Restful Api ，並以 Hibernate 框架實現 JPA ； 資料庫則選用 MySQL

## FrontEnd
前端部分以 Vue 搭建
